R2-E notes: "Gallery" layout — 36px rarity-colored icon squares centered horizontally, with 9px mod name captions below each. Labels wrap to 2 lines for long names (e.g., "Ext. Medium Mag I" and "Shotgun Choke II" both wrap neatly). Each item uses flex: 1; max-width: 90px so slots distribute evenly regardless of count (2-slot weapons spread out, 4-slot weapons are tighter). Empty slots use dashed-border icons with slot type label ("Tech Mod", "Stock"). On mobile, mod names use shorter forms to stay readable at the narrower width. This layout keeps text visible without hover/tap while maintaining visual scannability from the larger icons.
