Design notes: Crafting filters get their own dedicated sub-bar below the main filters — visually distinct with checkbox styling instead of pills. This keeps the main filter bar unchanged while giving crafting its own clear space. The auto-enable state (Mod Components forcing Mech on) is shown with a dimmed/forced visual + hint text. Mods render as ultra-compact rarity-colored dot badges — takes minimal vertical space. Build name appears inline as a tag in the weapon meta row (next to rarity badge). Empty slots shown as dim "Slot: —" text. Mobile uses abbreviated labels (Mech, Mod, KC, HG).
