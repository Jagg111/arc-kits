R2-D notes: Mod names are completely removed from the card layout — only 30px rarity-colored icon squares remain. This cuts ~40-50px of vertical height per weapon block compared to R2-A's vertical slot list. On desktop, hovering any icon shows a tooltip with the mod name (one is force-shown via .demo-visible class on the first Renegade muzzle). On mobile, tapping an icon toggles the .active class (one is pre-set on the mobile Renegade's Stock II icon). Empty slots use a dashed-border placeholder with a dash icon. The "no guide" state is handled with a simple italic text line.
